Calculadora en Java con Interfaz Gráfica

Una calculadora funcional con interfaz gráfica desarrollada en Java. Permite realizar operaciones básicas (+, -, *, /)

Tecnologías Utilizadas
	-Lenguaje: Java
	-Librerías: 
		~ javax.swing (Para la interfaz gráfica)
		~ java.awt (Para el manejo de eventos y diseño)

Requisitos
1. Requisitos:
   - Java jdk 20 o superior 

2. Ejecución:
   Ejecutar el proyecto a travez de .jar 


Características

	Operaciones básicas con sus botones
		Suma
		Resta
		Multiplicación
		Division

	Manejo de decimales (Boton de punto decimal[.]

	Botón de limpieza (C)

	Validación de operaciones inválidas (ej: división por cero)


Cómo Usar
	Ingresa números con los botones 0 al 9

	Selecciona un operador (+, -, *, /)

	Ingresa el segundo numero

	Presiona = para ver el resultado

	Usa C para borrar todo

Ejemplo:
	5 + 3 = → Muestra 8



Limitaciones
	No soporta operaciones encadenadas (2 + 3 * 4)

	No maneja paréntesis ni operaciones complejas

	El campo de texto no es editable manualmente
